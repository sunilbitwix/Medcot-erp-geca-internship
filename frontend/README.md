This repository contains the frontend application for the Med-Cotton ERP System, designed to manage operations of a medical cotton manufacturing business through a modern and responsive web interface.

The frontend communicates with the ERP backend using REST APIs and provides dashboards and management modules for inventory, production, users, and sales.

Built using React and Vite for fast development, clean structure, and optimized performance.

---

## Features

- Admin dashboard interface
- Inventory management UI
- Production tracking screens
- Sales and order management
- Role-based UI rendering
- REST API integration
- Responsive layout

---

## Tech Stack

- React.js
- Vite
- JavaScript (ES6+)
- Axios
- React Router
- CSS / Tailwind CSS

---

## Getting Started

### Install dependencies

```bash
npm install

## Run development server
npm run dev

## Build for production
npm run build
```
